# `krabkrab setup`

Initialize `~/.krabkrab/krabkrab.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
krabkrab setup
krabkrab setup --workspace ~/.krabkrab/workspace
```

To run the wizard via setup:

```bash
krabkrab setup --wizard
```
