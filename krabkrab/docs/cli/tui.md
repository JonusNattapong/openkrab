# `krabkrab tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
krabkrab tui
krabkrab tui --url ws://127.0.0.1:18789 --token <token>
krabkrab tui --session main --deliver
```
